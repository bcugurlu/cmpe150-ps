**Q2) Letter grade calculator**

```py
def letter_grade_generator(m1, m2, proj, hw, cw, q, f, a):
    score = 0.15 * m1 + 0.2 * m2 + 0.08 * proj + 0.06 * hw + 0.06 * cw + 0.1 * q + 0.35 * f
    if a / 28 >= 0.8:
        score += 2
    print(score)
    if score >= 85:
        return "A"
    elif 65 <= score < 85:
        return "B"
    elif 40 <= score < 65:
        return "C"
    elif 20 <= score < 40:
        return "D"
    else:
        return "F"


mt1 = int(input())
```

**Q3) Simple ATM**

```py
balance = 45178

if pin != true_pin:
    "Incorrect PIN, goodbye!"

        print(balance)
    elif op == 2:
        wd_request = int(input("Withdrawal request: "))

            print("Your withdrawal request is higher than the withdrawal limit. Goodbye!")

            print(balance-wd_request, "TRY")
```

**Q5) Prime or not**

```py
num = int(input())
prime = True

for i in range(2, num):
    if num%i == 0:
        prime = False

if prime:
    print("Prime")
else:
    print("Not prime")
```
